### 소개
- SpotHub은 지도서비스를 활용하여 사용자들이 서로 쓰레기통, 개방화장실, 폐건전지 수거함, 의류수거함 위치를 공유하는 서비스입니다.

## ❓ 주요 기능

- **위치 공유:** 사용자들은 쓰레기통, 개방화장실 등의 위치를 지도에 핀으로 표시하고 해당 장소에 대한 정보를 공유할 수 있습니다.
- **실시간 업데이트:** 지도에 표시된 정보는 실시간으로 업데이트되어 사용자들에게 최신 정보를 제공합니다.
